# SmartNote 사용자 스토리 v0.2

> 인증: 카카오 로그인 | AI: Llama | 휴지통 추가

**우선순위 기준**
- `Must` — 필수, MVP에 포함
- `Should` — 권장, Phase 2 이후
- `Could` — 선택, 여유 있을 때

---

## Epic 1 — 인증

### [AU-01] 카카오 로그인 `Must`

**사용자로서** 카카오 계정으로 로그인하고 싶다.
**나는 원한다** 별도 회원가입 없이 카카오 로그인만으로 서비스를 사용하기를.
**그래서** 빠르게 시작할 수 있다.

**인수 조건**
1. 카카오 로그인 버튼을 탭하면 카카오 인증 화면이 열린다
2. 로그인 성공 시 홈 화면으로 이동한다
3. 신규 사용자는 기타 그룹이 자동 생성된다
4. 재방문 시 자동 로그인된다
5. 로그아웃 기능이 설정에 존재한다

---

## Epic 2 — 알람 관리

### [AL-01] 알람 그룹 생성 `Must`

**사용자로서** 알람을 그룹 단위로 만들고 싶다.
**나는 원한다** 직장/일상처럼 상황별로 묶어 관리하기를.
**그래서** 한 번에 여러 알람을 제어할 수 있다.

**인수 조건**
1. 그룹 이름과 색상을 입력하고 저장할 수 있다
2. 기타 그룹은 삭제할 수 없다
3. 그룹 목록이 생성 순서로 표시된다
4. 그룹 수정/삭제가 가능하다

---

### [AL-02] 알람 그룹 ON/OFF 토글 `Must`

**사용자로서** 그룹 토글 버튼으로 그룹 전체를 끄고 켜고 싶다.
**나는 원한다** 휴일에 직장 알람을 한 번에 비활성화하기를.
**그래서** 매번 개별 알람을 끄지 않아도 된다.

**인수 조건**
1. 그룹 목록 화면에서 각 그룹 옆에 토글 버튼이 보인다
2. 토글 OFF 시 해당 그룹의 모든 알람이 비활성화된다
3. OFF 상태의 그룹과 알람 카드는 dimmed 처리된다
4. 토글 ON 복구 시 개별 알람의 기존 상태가 유지된다
5. 토글 변경은 즉시 반영된다

---

### [AL-03] 알람 생성 `Must`

**사용자로서** 시간, 반복 요일, 소리, 그룹을 설정해 알람을 만들고 싶다.
**그래서** 정해진 시간에 알람이 울린다.

**인수 조건**
1. 시간을 HH:mm 형식으로 설정할 수 있다
2. 반복 요일을 개별 선택할 수 있다 (일~토)
3. 소리 설정은 마지막 사용한 값이 디폴트로 적용된다
4. 진동 ON/OFF를 설정할 수 있다
5. 그룹을 선택하지 않으면 기타 그룹에 자동 배정된다

---

### [AL-04] 빠른 알람 `Must`

**사용자로서** 자연어로 알람을 빠르게 만들고 싶다.
**그래서** 메모 없이 즉시 알람만 등록할 수 있다.

**인수 조건**
1. 알람 탭 상단에 빠른 알람 입력창이 표시된다
2. "30분 후", "내일 9시 기상" 등을 입력하면 시간이 파싱된다
3. 파싱 결과를 확인 화면에서 볼 수 있다
4. 그룹을 선택한 뒤 저장할 수 있다
5. 파싱 실패 시 안내가 표시된다

---

### [AL-05] 조건부 그룹 자동 OFF `Should`

**사용자로서** 비출근일에 직장 그룹이 자동으로 꺼지길 원한다.
**그래서** 휴일에 직접 끄지 않아도 된다.

**인수 조건**
1. 캘린더에서 날짜별 출근 여부를 설정할 수 있다
2. 비출근일 자정에 직장 그룹이 자동으로 OFF된다
3. 다음 출근일 자정에 직장 그룹이 자동으로 ON된다

---

## Epic 3 — 메모

### [ME-01] 메모 작성 `Must`

**사용자로서** 제목과 본문을 자유롭게 작성해 저장하고 싶다.
**그래서** 나중에 볼 수 있도록 기록할 수 있다.

**인수 조건**
1. 제목과 본문을 입력할 수 있다
2. 저장 버튼을 누르면 메모 목록에 추가된다
3. 메모 목록은 최신순으로 정렬된다
4. 작성 시각이 자동으로 기록된다

---

### [ME-02] 위치 자동 태깅 `Should`

**사용자로서** 메모 작성 시 현재 위치가 자동으로 기록되길 원한다.
**그래서** 어디서 작성했는지 나중에 확인할 수 있다.

**인수 조건**
1. 메모 저장 시 GPS 좌표가 자동 수집된다
2. 역주소로 변환되어 메모에 표시된다 (예: 강남역 근처)
3. 위치 권한을 거부한 경우 위치 태그 없이 정상 저장된다
4. 위치 정보가 메모 카드 하단에 표시된다

---

### [ME-03] AI 메모 정리 `Must`

**사용자로서** 급하게 쓴 메모를 AI가 정리해주길 원한다.
**그래서** 나중에 다시 봐도 이해할 수 있다.

**인수 조건**
1. 메모 저장 후 백그라운드에서 Llama AI 정리가 자동 시작된다
2. AI 정리 완료 전에는 처리 중 상태가 표시된다
3. 토글 버튼으로 원문 / AI 정리를 언제든지 전환할 수 있다
4. AI 정리 실패 시 원문만 표시되고 재시도 버튼이 보인다
5. 재분석 버튼으로 수동으로 AI 재처리를 요청할 수 있다

---

### [ME-04] 자동 알람 감지 `Must`

**사용자로서** 메모에 날짜/시간을 쓰면 자동으로 알람을 제안받고 싶다.
**그래서** 메모를 쓰다 보면 자연스럽게 알람이 생긴다.

**인수 조건**
1. 메모 본문에서 날짜/시간 표현이 감지되면 알람 제안 배지가 표시된다
2. 추가 버튼을 탭하면 그룹 선택 화면이 나타난다
3. 그룹을 선택 후 확인하면 알람이 생성된다
4. 취소를 탭하면 제안이 사라지고 메모만 유지된다

---

## Epic 4 — 홈

### [HO-01] 자연어 입력 & AI 분류 `Must`

**사용자로서** 무엇이든 자유롭게 입력하면 AI가 알아서 분류해주길 원한다.
**그래서** 앱을 전환하거나 카테고리를 고민하지 않아도 된다.

**인수 조건**
1. 홈 화면에 자연어 입력창이 표시된다
2. 입력 후 Llama가 일정/알람/할일/메모/나중에/언젠가로 분류한다
3. 분류 결과를 확인 화면에서 볼 수 있다
4. 분류가 잘못된 경우 수동으로 변경할 수 있다
5. 확인 후 해당 컬렉션에 저장된다

---

## Epic 5 — 나중에 알려줘

### [LA-01] 나중에 알려줘 등록 & 알림 `Must`

**사용자로서** 지금 당장은 아니지만 잊기 싫은 항목을 저장하고 싶다.
**그래서** 원하는 시간에 다시 알림을 받을 수 있다.

**인수 조건**
1. 항목을 저장하고 알림 시간을 선택할 수 있다
2. 알림 시간 옵션: 오늘 저녁 / 이번 주말 / 다음주 / 직접 설정
3. 알림 수신 후 완료 또는 미루기를 선택할 수 있다
4. 미루기 선택 시 새 시간을 설정할 수 있다

---

## Epic 6 — 언젠가 리스트

### [SO-01] 언젠가 리스트 등록 & 분류 `Must`

**사용자로서** 언젠가 하고 싶은 것들을 모아두고 싶다.
**그래서** 잊지 않고 꺼내볼 수 있다.

**인수 조건**
1. 항목을 입력하면 Llama가 카테고리를 자동 분류한다
2. 카테고리: 여행 / 배움 / 구매 / 기타
3. 수동으로 카테고리를 변경할 수 있다
4. 카테고리 탭으로 필터링할 수 있다
5. 즐겨찾기 설정이 가능하다

---

## Epic 7 — 캘린더

### [CA-01] 캘린더 기본 뷰 `Must`

**사용자로서** 월간 캘린더에서 일정을 확인하고 싶다.
**그래서** 오늘과 이번 달 일정을 한눈에 볼 수 있다.

**인수 조건**
1. 월간 캘린더가 표시된다
2. 오늘 날짜가 강조 표시된다
3. 알람/메모가 연동된 날짜에 dot 표시가 보인다
4. 날짜를 탭하면 해당 날의 일정 목록이 표시된다
5. 이전/다음 월로 이동할 수 있다

---

### [CA-02] 출근일 설정 `Should`

**사용자로서** 날짜별로 출근/비출근을 설정하고 싶다.
**그래서** 알람 그룹 자동 OFF 기능과 연동할 수 있다.

**인수 조건**
1. 날짜를 탭하면 출근/비출근 설정 옵션이 나타난다
2. 출근일은 캘린더에서 시각적으로 구분된다
3. 설정이 즉시 Firestore에 저장된다
4. 기본값은 평일 출근일, 주말 비출근일이다

---

## Epic 8 — 휴지통

### [TR-01] 항목 삭제 & 복원 `Must`

**사용자로서** 실수로 삭제한 항목을 복원하고 싶다.
**그래서** 중요한 데이터를 잃지 않을 수 있다.

**인수 조건**
1. 삭제된 항목은 30일간 휴지통에 보관된다
2. 휴지통에서 원래 위치로 복원할 수 있다
3. 휴지통에서 즉시 영구 삭제할 수 있다
4. 30일이 지나면 자동으로 영구 삭제된다
5. 휴지통 목록에서 남은 보관 기간이 표시된다

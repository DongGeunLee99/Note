# SmartNote — AI 데이터 처리 가이드 (v0.1)

> 개발자용 결정 기록. AI(LLM) 기능이 **어떤 데이터를 / 어디로 / 왜** 보내는지와 그 처리 원칙을 정리한다.
> 사용자용 개인정보 처리방침(Privacy Policy)은 배포 전 이 문서를 바탕으로 별도 작성한다.
> 최초 작성 2026-06-18. 관련 결정: AI 제공자 = Gemini Flash 무료.

---

## 1. 제공자 결정

| 항목 | 내용 |
|---|---|
| LLM 제공자 | **Google Gemini Flash (무료 등급)** |
| 호출 위치 | **Firebase Cloud Functions** (API 키는 서버 보관, 클라이언트 노출 금지) |
| 추상화 | `packages/shared/services/llamaService.ts` 인터페이스 뒤에 구현 → 제공자 교체 용이 |
| 대체 후보 | 온디바이스(Chrome 내장 AI), Gemini 유료(학습 제외), 타 무료 API |

> 선정 이유: 무료 + 자체 서버 불필요 + 다중 사용자 배포 전제. 규칙으로 불가능한 작업(자유서술 분류, 한국어 시간표현·오타, AM/PM 문맥 추론, 메모 요약)에 실제 LLM이 필요. 온디바이스는 해당 품질에 미달 위험 + 브라우저 제약.

---

## 2. AI 기능별 전송 데이터

| 기능 | 외부로 전송되는 데이터 | LLM 필요성 |
|---|---|---|
| 메모 **AI 정리**(요약/정돈) | 메모 본문 | 필수(생성형) |
| 홈 **입력 분류 + 구조화** | 홈 입력 텍스트 | 필요(문맥 추론) |
| 메모/입력 **시간 추출 + AM·PM** | 해당 텍스트 일부 | 애매한 경우만 |

- 한 입력당 **1회 호출로 구조화 JSON**(category / datetime / ampm / label) 수신 — 호출 수 최소화.
- 전송 = 처리를 위해 텍스트가 구글 서버로 감(클라우드 API의 본질).

---

## 3. 프라이버시 — 무료 등급의 핵심 사실

- **Gemini API 무료 등급**: 입력/출력이 구글 서비스 개선(모델 학습·사람 검토)에 **사용될 수 있음**.
- **유료 등급**: 학습에 사용하지 않음(전송은 동일).
- 소비자 앱(gemini.google.com 등)과 유사하게, 무료 등급은 데이터가 학습에 쓰일 수 있다는 점을 **사용자에게 고지**해야 함.

> ⚠️ 약관은 변경될 수 있으므로 도입·배포 시점에 **최신 약관 재확인** 필수.

---

## 4. 처리 원칙 / 완화책

1. **규칙 선거름**: 명확한 입력(예: "오전 7시")은 chrono/규칙으로 로컬 처리 → LLM 미호출. 애매할 때만 LLM.
2. **최소 전송**: 분류/시간추출은 필요한 텍스트 조각만 전송(불필요한 전체 본문 남용 지양).
3. **사용자별 호출 제한**: Cloud Functions에서 사용자당 호출 빈도 제한 → 무료 할당량 보호 + 남용 방지.
4. **키 보관**: API 키는 Cloud Functions 환경에만. 클라이언트/리포지토리에 노출 금지.
5. **결과 저장**: AI 정리 결과 등은 사용자 본인 Firestore 문서에 저장. 원문 외 별도 외부 보관 없음.
